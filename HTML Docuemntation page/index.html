img tag

This tag is used to insert an image. It's a self-closing tag. Insert the location of the image file

<img href=#>

main tag

This is used to specfy the main content of a website. This requires a closing tag.

<main> </main>

P tag

Defines a paragraph. This requires a closing tag.

<P> </p>

Header tag

Defines a header for a document or section. This requires a closing tag.

<header> </header>

Head tag

Contains metadata/information for the document. This requires a closing tag.

<head> </head>
